- Updated smali to `v1.3.0`

- Fixed decoding error when string ends with `%` [#280](https://github.com/iBotPeaches/Apktool/issues/280)
- Fixed decoding error when `<plurals>` contains a reference [#345](https://github.com/iBotPeaches/Apktool/issues/345)
- Fixed a broken res when decoding `<array>` with positional substitutions [#333](https://github.com/iBotPeaches/Apktool/issues/333)
