- Added 9patch image decoding support. [#112](https://github.com/iBotPeaches/Apktool/issues/112)
- Fixed support for APKs lacking res directory of resources.arsc file. ([#154](https://github.com/iBotPeaches/Apktool/issues/154), [#160](https://github.com/iBotPeaches/Apktool/issues/160))
- Changed default name of built file from `out.apk` to `<original_name>.apk`
- Added possibility to chose location of output APK when building.
